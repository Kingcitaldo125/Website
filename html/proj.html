S.S.U.R.G.E.

This is described by our professor as a DIY engine that we(about 20 of us total) made during the course of 2 semesters in Realtime Interactive Programming. Features include sound, shaders, third person controllers, animation, physics and much more.

Cabin in the Woods

This is a game project that I worked on during Computer Graphics II at Shawnee State University. All code was written in Javascript and GLSL. Simple game concept: walk around a map and pick up pages. Collect all of the pages to win the game. The game idea was loosely based on the Slender: The Nine Pages game.

Python Planet Defense


A game I made in Visual Studio with Python. Simple Concept: keep large asteroids away from your home planet. AI controllers help you accomplish said task

Chess

Chess made for Networking and Communications class at Shawnee State

Target Shooter

A target shooter game made in Unity3d with a group of fellow students at Shawnee State during the last 3 weeks of Realtime Interactive Programming Two. Based off of the PIT from Call of Duty Modern Warfare 2

Basic OS Kernel

An Operating System kernel made as a final project for Operating Systems One at Shawnee State. The video shows the capabilities of typing text into a shell, then the Kernel will read the text back after the 'Enter' key is pressed. Made in C and Assembly languages.
